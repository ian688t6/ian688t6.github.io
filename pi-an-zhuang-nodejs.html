Pi 安装 Node.js

Published: Sun 06 January 2019
By J.Ian

In Pi.

获取node.js package最新的稳定版本

$ wget http://node-arm.herokuapp.com/node_latest_armhf.deb

安装

$ sudo dpkg -i node_latest_armhf.deb

验证是否安装成功

$ node -v

结束
